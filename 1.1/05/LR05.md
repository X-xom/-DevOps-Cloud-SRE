### D5-01
Из less выход — q
Конвейер( | ) — это одна из главных "фишек" Linux. Он позволяет взять результат работы одной команды и передать его другой команде на вход.
Синтаксис: `команда1 | команда2 | команда3`

### D5-04 Какие клавиши использовала (/, n, q):
/ - для поиска 
n - переходит к следующему вхождению поискового термина
q - выход из less

### D5-06 Про -i/-n и поиск в less
i - команда не обращает внимания на большие и маленькие буквы
n - показывает не только строки с текстом, но и их номера
Работа в less:
/ - поиск вниз по тексту
? - поиск вверх по тексту
Чтобы листать результаты: n — следующее совпадение, N — предыдущее

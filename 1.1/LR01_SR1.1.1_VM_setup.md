## Цель работы

Создать собственную изолированную среду для экспериментов с Linux — установить виртуальную машину с серверным дистрибутивом.

## Задание (все подпункты)

Ниже — полный перечень подпунктов из задания. Отмечайте выполнение прямо в процессе.

1.  **Подготовка:**
    - Скачайте и установите на свой компьютер программу для виртуализации. Рекомендуется **Oracle VirtualBox** (бесплатная, кроссплатформенная). Ссылка для скачивания есть на официальном сайте.
    - Скачайте образ дистрибутива Linux. Рекомендуется **Ubuntu Server LTS** (последняя версия с долгосрочной поддержкой). Образ можно скачать с официального сайта ubuntu.com. Альтернатива: **Astra Linux Common Edition** (сайт astralinux.ru).

2.  **Создание виртуальной машины:**
    - Запустите VirtualBox.
    - Нажмите "Создать" (Create).
    - **Имя:** `Linux-Ubuntu` (или любое другое).
    - **Тип:** Linux.
    - **Версия:** Ubuntu (64-bit).
    - **Объём памяти:** Выделите не менее 2048 МБ (2 ГБ).
    - **Жёсткий диск:** Создать новый виртуальный жёсткий диск. Тип — VDI (VirtualBox Disk Image). Формат хранения — динамический. Размер — не менее 20 ГБ.

3.  **Установка операционной системы:**
    - Выберите созданную машину и нажмите "Запустить".
    - Укажите путь к скачанному образу дистрибутива.
    - Следуйте инструкциям установщика:
        - Выберите язык (English рекомендуется для привыкания к терминологии, можно Русский).
        - Выберите раскладку клавиатуры.
        - **Настройка сети:** Обычно DHCP настраивает всё автоматически. Запишите IP-адрес, если он отобразится.
        - **Настройка пользователя:** Придумайте имя компьютера (например, `ubuntu-vm`), имя пользователя (например, `student`) и **пароль**. **ЗАПОМНИТЕ ПАРОЛЬ!**
        - **Разметка диска:** Для первого раза выберите самый простой вариант — "Use An Entire Disk" (использовать весь диск) и просто подтвердите.
        - **Установка ПО:** Можно снять галочку с "Ubuntu Server extras" (чтобы ускорить установку), но убедитесь, что **OpenSSH server** выбран для установки. Это позволит подключаться к ВМ по SSH.
    - Дождитесь окончания установки и перезагрузки.

4.  **После установки:**
    - Войдите в систему под своим пользователем (в консоли виртуальной машины).
    - Выполните команду `ip a`. Найдите строку, похожую на `inet 192.168.x.x`. Это IP-адрес вашей ВМ. Запишите его.
    - Выключите виртуальную машину: `sudo poweroff`.

5.  **Проверка SSH-подключения (с основного компьютера):**
    - Убедитесь, что виртуальная машина выключена.
    - В настройках сети виртуальной машины в VirtualBox выберите тип подключения "Сетевой мост" (Bridged Adapter) или "NAT" с пробросом портов (сложнее). Для первого раза проще всего использовать "Сетевой мост", если в вашей сети раздаётся DHCP.
    - Запустите виртуальную машину снова.
    - Откройте терминал на вашем основном компьютере и попробуйте подключиться:
        ```bash
        ssh ваш_пользователь@IP-адрес_ВМ
        ```
        (Например: `ssh student@192.168.1.50`)
    - Если подключение успешно, вы великолепны!

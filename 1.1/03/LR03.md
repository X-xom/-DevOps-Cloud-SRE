### Из Лекции 1.1.3.

mkdir — используется для создания новых каталогов.
touch — служит для создания пустых файлов или обновления временных меток существующих файлов.
cp — предназначена для копирования файлов и каталогов.
mv — выполняет перемещение или переименование файлов и каталогов.
rm — удаляет файлы. Для удаления каталогов применяется с опцией -r.
rmdir — удаляет только пустые каталоги.

### Ответ на задание 70 (Проверь себя)

1. Что символизирует корень файловой системы в Linux?  
Корень файловой системы обозначается символом /. Он является вершиной иерархической структуры, к которой монтируются все разделы дисков и устройства.

2. Для чего нужна папка /etc? А /var/log?  
Каталог /etc предназначен для хранения конфигурационных файлов операционной системы и прикладного программного обеспечения. Каталог /var/log содержит файлы журналов, в которые записываются события системы и работающих служб.

3. Какая команда покажет полный путь к текущей папке? А покажет содержимое, включая скрытые файлы?  
Команда pwd отображает полный путь к текущему рабочему каталогу. Команда ls -a выводит список содержимого каталога, включая скрытые файлы.

4. Чем отличается абсолютный путь от относительного? Приведи примеры.  
Абсолютный путь задает местоположение файла или каталога, начиная от корневого каталога. Относительный путь задает местоположение относительно текущего рабочего каталога.

5. Ты находишься в /home/user. Как одной командой создать структуру work/project/2024?  
Для создания данной иерархической структуры каталогов используется команда mkdir -p work/project/2024.

6. Как скопировать папку data в папку backup? Почему просто cp data backup не сработает?  
Для копирования каталога data в каталог backup необходимо использовать команду cp -r data backup. Без опции -r команда cp предназначена только для копирования отдельных файлов.

7. В чем опасность команды rm -rf /? Сформулируй "золотое правило", которое поможет избежать беды.  
Команда rm -rf / является критически опасной, так как она рекурсивно и принудительно удаляет все данные в корневом каталоге, что приводит к полной неработоспособности операционной системы.  
Золотое правило безопасности: Перед выполнением команды rm -rf необходимо убедиться в правильности текущего каталога с помощью команды pwd и проверить список удаляемых объектов командой ls.

### D7-01 linux_final_review.txt

1. Что такое ядро Linux? Какова его роль?
Ядро Linux — это центральная часть операционной системы, которая управляет аппаратными ресурсами компьютера и обеспечивает взаимодействие между программами и оборудованием. Оно отвечает за распределение памяти, планирование задач, работу с драйверами и файловой системой.
2. Что такое дистрибутив? Назовите 3 примера и кратко опишите их назначение.
Дистрибутив — это набор из ядра Linux, системных утилит, менеджера пакетов и прикладного программного обеспечения, собранный в готовую к использованию операционную систему.
- Ubuntu — популярный дистрибутив для начинающих и настольных компьютеров.
- CentOS — стабильный дистрибутив для серверов (ныне заменен на Rocky Linux/AlmaLinux).
- Kali Linux — специализированный дистрибутив для тестирования на проникновение и кибербезопасности.
3. Зачем нужен SSH? Напишите полную команду для подключения к серверу с IP 10.0.0.5 от пользователя admin.
SSH нужен для удаленного безопасного управления сервером по зашифрованному каналу.
Команда: ssh admin@10.0.0.5
4. Перечислите 5 основных команд для навигации по файловой системе. Что делает каждая?
- pwd — показывает текущий рабочий каталог.
- ls — выводит список содержимого каталога.
- cd — меняет текущий каталог.
- cd .. — переходит на уровень выше.
- cd ~ — переходит в домашний каталог пользователя.
5. Какая команда показывает содержимое файла постранично?
Команда less (например: less filename.txt).
6. Как создать новую папку с именем test_dir?
mkdir test_dir
7. Как скопировать файл file.txt в папку backup?
cp file.txt backup/
8. Как переименовать файл old.txt в new.txt?
mv old.txt new.txt
9. Как удалить папку old_data со всем её содержимым? Напишите команду и главное правило безопасности перед её выполнением.
Команда: rm -rf old_data
Главное правило безопасности: перед выполнением команды необходимо выполнить pwd, чтобы убедиться, что вы находитесь в правильном каталоге, и ls, чтобы проверить, что именно будет удалено.
10. Как найти все строки, содержащие слово "warning" в файле app.log, и показать их с номерами строк?
grep -n "warning" app.log
11. Что такое конвейер (|)? Приведите пример.
Конвейер (|) — это механизм передачи вывода одной команды на вход другой команде.
Пример: cat file.txt | grep "error" — сначала выводит содержимое файла, затем передает его в grep для поиска слова "error".
12. Как получить подробную справку по команде ls? А краткую?
Подробная справка: man ls
Краткая справка: ls --help

### D7-03 linux_cheatsheet.txt
ШПАРГАЛКА

НАВИГАЦИЯ
pwd                      - где я сейчас
ls                       - что тут есть
ls -a                    - что тут есть (показать скрытые)
cd папка                 - зайти в папку
cd ..                    - выйти на уровень вверх
cd ~                     - вернуться домой

ФАЙЛЫ И ПАПКИ
touch имя                - создать пустой файл
mkdir имя                - создать папку
cp что куда              - скопировать
mv что куда              - переместить/переименовать
rm файл                  - удалить файл
rm -r папка              - удалить папку
rm -rf папка             - удалить папку (осторожно!!!)

ПОСМОТРЕТЬ ФАЙЛ
cat файл                 - показать весь файл
less файл                - листать файл (q выход, / поиск)
head файл                - показать начало
tail файл                - показать конец
tail -f файл             - смотреть изменения в реальном времени

GREP (ПОИСК)
grep слово файл          - найти слово в файле
grep -i слово файл       - найти (плевать на регистр)
grep -n слово файл       - найти и показать номера строк
grep -v слово файл       - найти строки без этого слова

КОНВЕЙЕР (|)
команда1 | команда2      - результат первой скормить второй
ls | grep txt            - найти все txt файлы в текущей папке

ЛОГИ
/var/log/                - тут все логи
sudo tail -f auth.log    - следить за попытками входа
sudo grep sshd auth.log  - найти все про ssh

ПОМОЩЬ
man команда              - полная инструкция
команда --help           - краткая справка
